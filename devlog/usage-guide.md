# PasteEcho — 使用说明

## 软件用途

PasteEcho 是一款 macOS 菜单栏剪贴板管理工具。它自动记录你最近复制（Cmd+C）的所有文字和图片，方便随时回溯和再次使用。

## 如何启动

1. 双击 `PasteEcho.app` 启动应用
2. 应用在后台运行，菜单栏右上角会出现一个剪贴板图标
3. 应用不显示 Dock 图标（纯菜单栏 App）

## 如何查看历史粘贴板

- **点击菜单栏图标**：弹出历史记录面板（380×520）
- 卡片按时间降序排列（最新的在最上面）
- 置顶的卡片始终排在最前面
- 文字显示内容预览（最多2行），图片显示缩略图

## 如何搜索

- 在面板顶部的搜索框中输入关键词
- 列表实时过滤匹配的文字内容
- 点击清除按钮（×）可清空搜索
- 图片不参与搜索

## 如何使用历史记录

### 重新粘贴
- 点击任意卡片，内容即复制到系统剪贴板
- 在目标 App 中按 Cmd+V 粘贴
- 点击后卡片右侧显示绿色对勾表示已复制

### 置顶
- 鼠标悬停在卡片上，点击图钉按钮
- 置顶后卡片有浅蓝色背景，始终排在顶部
- 再次点击取消置顶

### 删除
- 鼠标悬停在卡片上，点击垃圾桶按钮
- 删除后不可恢复（关联的图片文件也同步删除）

## 如何唤起窗口

- **菜单栏点击**：点击菜单栏 PasteEcho 图标
- **全局快捷键**：在任意 App 中按 `Cmd + Shift + V` 呼出/关闭面板
- **点击外部关闭**：点击面板外的任意区域，面板自动关闭

## 设置

点击面板右上角的齿轮图标打开设置窗口：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| Retention Period | 保留天数（1/3/5天），过期自动删除 | 3 Days |
| Max Items | 最大条数（50-500），超出后删除最旧的 | 200 |
| Launch at Login | 开机自动启动 | 开启 |

> 注意：置顶的条目不会被自动清理。

## 自动监听机制

- 应用以 0.5 秒间隔轮询系统剪贴板
- 检测到变化后自动捕获文字或图片
- 连续复制相同内容不会产生重复记录
- 捕获的内容即时保存到磁盘

## 数据存储位置

所有数据存储在本地，不上传任何网络：

```
~/Library/Application Support/PasteEcho/
├── items.json       # 历史记录元数据
├── settings.json    # 用户设置
└── Images/          # 图片文件
```

JSON 损坏时有自动备份机制（`.bak` 文件）。

## 系统要求

- macOS 13 (Ventura) 或更高版本
- Intel 或 Apple Silicon Mac

## 已知问题

1. 首次启动需授予辅助功能权限（如使用未来版本的全局监听）
2. 图片缩略图在极端尺寸下可能略微模糊
3. 搜索不支持中文分词（仅全文匹配）
