# PasteEcho — 产品需求文档 (PRD)

## 产品定位

一款运行在 macOS 上的轻量级历史剪贴板管理工具，常驻菜单栏，自动记录用户复制的内容，方便随时回溯和再次使用。

## 目标用户

macOS 普通用户，不需要技术背景。

## 功能需求

### F1. 剪贴板自动捕获

- 自动监听系统剪贴板变化
- 支持捕获**文字**和**图片**两种内容类型
- 去重：连续复制相同内容不重复记录
- 在后台静默运行，不影响用户正常操作

### F2. 历史记录展示

- 以**卡片列表**形式展示所有历史记录
- 按**时间降序**排列（最新的在最上面）
- 置顶的卡片始终排在最前面
- 文字卡片显示：文本图标 + 内容预览（最多2行）
- 图片卡片显示：**48×48 缩略图** + "图片"标签
- 每条记录显示**相对时间**（如 "3分钟前"、"昨天 14:30"）
- 列表为空时显示友好的空状态提示

### F3. 卡片操作

- **点击卡片**：将该内容重新写入系统剪贴板，用户可在任意 App 中 Cmd+V 粘贴
- **置顶/取消置顶**：将重要内容固定在列表顶部
- **删除**：移除不需要的记录
- 粘贴成功后给予视觉反馈（如 "已复制" 提示）

### F4. 搜索

- 顶部搜索框，输入关键词实时过滤
- 仅搜索文字内容（图片不参与搜索）
- 搜索框支持一键清除

### F5. 存储管理

- 支持设置保留天数：**1天 / 3天 / 5天**
- 支持设置最大条数上限（默认 **200** 条）
- 过期或超限的条目自动清理（置顶条目不会被自动清理）
- 关联的图片文件同步删除

### F6. 菜单栏

- 顶部菜单栏显示应用图标
- 点击图标弹出历史面板
- 点击面板外部区域自动关闭
- 应用不显示 Dock 图标（纯菜单栏 App）

### F7. 全局快捷键

- 默认快捷键：**Cmd + Shift + V**
- 在任意 App 中按下可呼出/关闭历史面板
- 无需辅助功能权限

### F8. 开机自启动

- 提供开关选项
- 开启后 Mac 开机自动在后台运行
- 使用 macOS 原生 `SMAppService` 实现

### F9. 设置窗口

- 独立设置窗口，通过面板中的齿轮图标打开
- 可配置项：
  - 保留天数（1天 / 3天 / 5天 单选）
  - 最大条数（数值输入）
  - 开机启动（开关）

## 非功能需求

### N1. 性能
- 剪贴板轮询间隔 ≤ 0.5 秒，CPU 占用可忽略
- 卡片列表滑动流畅（LazyVStack 懒加载）
- 存储读写 debounce 防抖，避免频繁磁盘 I/O

### N2. 隐私
- 所有数据存储在本地 `~/Library/Application Support/PasteEcho/`
- 不上传任何数据到网络
- 不排除特定 App（记录所有复制内容）

### N3. 兼容性
- 最低支持 macOS 13 (Ventura)
- 适配系统深色/浅色模式

### N4. 可靠性
- 图片数据独立文件存储，单张图片损坏不影响其他数据
- JSON 文件写入使用原子操作，防止写入中断导致数据丢失

## UI 设计原则

- **简洁直观**：最小化操作步骤，一目了然
- **淡蓝色主调**：`#4D99F2` 作为品牌色
- **原生风格**：遵循 macOS Human Interface Guidelines
- **信息层级清晰**：标题 > 内容预览 > 时间 > 操作按钮
