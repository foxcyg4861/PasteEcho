# PasteEcho — 设计规范文档

## 品牌色彩

| 颜色 | 色值 | 用途 |
|------|------|------|
| 主色 (Blue) | `#4D99F2` | 品牌色、强调元素、按钮 |
| 浅蓝背景 | `#4D99F2` @ 8% 透明度 | 置顶卡片背景 |
| 浅蓝边框 | `#4D99F2` @ 20% 透明度 | 选中态边框 |
| 卡片背景 | `NSColor.controlBackgroundColor` | 跟随系统深浅色模式 |
| 次要文字 | `Color.secondary` | 时间戳、提示文字 |
| 危险操作 | `Color.red.opacity(0.7)` | 删除按钮 |

## 交互规范

### 卡片

- **默认态**：卡片背景透明，显示内容和时间
- **悬停态**：显示操作按钮（置顶、删除），鼠标指针变为手型
- **点击**：复制内容到剪贴板，短暂显示 "已复制" 图标反馈
- **置顶态**：浅蓝色背景，固定在列表顶部

### 搜索

- 输入即搜索，无延迟
- 空搜索框显示全部
- 搜索无结果时显示 "无匹配结果"
- 清除按钮仅在有输入时显示

### 设置

- 点击齿轮图标打开独立设置窗口
- 修改即时保存生效
- 关闭窗口不丢失未保存内容

### 粘贴反馈

- 点击卡片后，卡片右侧短暂显示绿色对勾 (0.5秒)
- 对勾消失后恢复正常

## 深色模式

- 卡片背景自动跟随 `controlBackgroundColor`（系统自动适配）
- 文字颜色使用 `.primary` / `.secondary`（系统自动适配）
- 淡蓝色主色在深色背景下略调亮至 `#5DADFF`（可选优化）
- 卡片边框在深色模式下透明度调整为 20%

## 动画

- 弹出面板出现：系统默认 popover 动画
- 卡片出现/消失：`.transition(.opacity.combined(with: .move(edge: .top)))`
- 搜索过滤：无动画（即时过滤）
- 已复制图标：`.scaleEffect` 弹入弹出 0.3s
